#### # Instructions

#### Given the position of two queens on a chess board, indicate whether or not they are positioned so that they can attack each other.  
####   
#### In the game of chess, a queen can attack pieces which are on the same row, column, or diagonal.  
####   
#### A chessboard can be represented by an 8 by 8 array.  
####

####   
#### You are also able to answer whether the queens can attack each other.

#include "queen_attack.h"
#include <stdexcept>
#include <cctype>
#include <string>

using namespace std;

namespace queen_attack {

    bool can_attack(const string& white, const string& black) {
        // Parse white queen position
        if (white.empty() || black.empty()) {
            throw invalid_argument("Invalid queen position");
        }
        
        // Validate and convert white position
        char w_col = tolower(white[0]);
        if (w_col < 'a' || w_col > 'h') throw invalid_argument("Invalid column");
        int w_col_idx = w_col - 'a';
        
        int w_row;
        try {
            w_row = 8 - stoi(white.substr(1));
        } catch (...) {
            throw invalid_argument("Invalid row");
        }
        if (w_row < 0 || w_row > 7) throw invalid_argument("Row out of bounds");

        // Validate and convert black position 
        char b_col = tolower(black[0]);
        if (b_col < 'a' || b_col > 'h') throw invalid_argument("Invalid column");
        int b_col_idx = b_col - 'a';
        
        int b_row;
        try {
            b_row = 8 - stoi(black.substr(1));
        } catch (...) {
            throw invalid_argument("Invalid row");
        }
        if (b_row < 0 || b_row > 7) throw invalid_argument("Row out of bounds");

        // Check attack conditions
        return (w_col_idx == b_col_idx) ||  // Same column
               (w_row == b_row) ||          // Same row
               (abs(w_col_idx - b_col_idx) == abs(w_row - b_row)); // Diagonal
    }

}  // namespace queen_attack

The implementation:

1. Adds the `can_attack` function declaration to the header with proper includes
2. Implements position validation and parsing in the cpp file:
   - Converts column letters (a-h) to 0-7 indices
   - Converts numeric rows (1-8) to 0-7 indices (with 0 being the top row)
   - Throws `invalid_argument` for any invalid input
3. Checks three possible attack conditions:
   - Same column
   - Same row
   - Same diagonal (absolute differences between columns and rows are equal)

The code handles all edge cases and follows chess rules for queen movement. The example from the problem description (c5 vs f2) would be parsed as positions (2,3) and (5,6) which are on the same diagonal (difference of 3 columns and 3 rows).
